# Naval Thread #1: 如何致富（不靠運氣）

**Source:** Naval Ravikant - How to Get Rich (Without Getting Lucky)
**Format:** Twitter Thread
**Account:** @0xjh1nr3sh

---

## Thread Content

**1/ 🧵**
Naval 的「如何致富」是史上最強的財富哲學。

我花了一週翻譯整理，這 15 條將改變你對錢的看法。

（收藏起來，反覆讀）

**2/**
💰 追求財富，而非金錢或地位

財富 = 睡覺時也在賺錢的資產
金錢 = 轉移財富的工具
地位 = 社會階層中的排名

你要的是財富，不是其他兩個。

**3/**
🎯 財富的目的是自由

不是買名牌、開跑車、遊艇環遊世界。
那些東西很快就會無聊。

財富的唯一目的：
成為自己的主人。

**4/**
⚔️ 兩種遊戲

財富遊戲 = 正和遊戲（大家一起變富）
地位遊戲 = 零和遊戲（你上我就得下）

創造財富的人，永遠會被玩地位遊戲的人攻擊。

遠離地位遊戲。

**5/**
🌍 每個人都能變富

200 年前沒有抗生素、汽車、電力、iPhone。
這些發明讓全人類都更富有了。

我寧願當今天的窮人，也不當路易十四時代的貴族。

這就是財富創造的力量。

**6/**
🎲 運氣的四種類型

1️⃣ 盲目運氣 — 純粹命運
2️⃣ 努力運氣 — 拼命製造機會
3️⃣ 準備運氣 — 技能讓你看見機會
4️⃣ 獨特運氣 — 你的個人品牌吸引機會

第四種最重要。

**7/**
🤿 獨特運氣的例子

假設你是世界上最厲害的深海潛水員。

有人發現沉船寶藏，但自己拿不到。
他們會來找你，給你一半。

他們的運氣 → 變成你的運氣。

這不是運氣，這是你創造的。

**8/**
🎯 在 1000 個平行宇宙中...

你要在 999 個裡面都變富。
不是只在 50 個靠運氣變富的宇宙裡。

把運氣從等式中移除。
靠技能、判斷力、獨特性。

**9/**
🧠 變成「會賺錢的那種人」

如果我失去所有錢，把我丟到任何英語國家的街頭。

5-10 年內我會再次變富。

因為這是一種技能。
任何人都能學會。

**10/**
⚡ 槓桿的秘密

財富來自槓桿。三種類型：

1️⃣ 勞力 — 讓別人為你工作
2️⃣ 資本 — 用錢生錢
3️⃣ 產品 — 代碼和媒體，零邊際成本複製

第三種最強大，也最公平。

**11/**
💻 新時代的槓桿

代碼和媒體是新富人的槓桿。

一段代碼可以服務百萬用戶
一條推文可以影響百萬人
一個 podcast 可以播放百萬次

而且不需要任何人的許可。

**12/**
🔑 特定知識

讓你致富的不是通用技能。

是「特定知識」— 
無法被訓練的知識
感覺像玩樂的技能
是你的天賦和獨特興趣的交集

**13/**
🎮 把工作變成遊戲

如果某件事對你來說像遊戲，對別人來說像工作 —

你就贏定了。

因為你會玩得更久、更投入、更有創意。

**14/**
📚 閱讀的力量

讀你愛讀的書，直到你愛上閱讀。

然後什麼都讀。

讀書不是為了「完成」書。
是為了找到改變你思維的想法。

**15/**
🏆 總結

• 追求財富，不是地位
• 創造價值，不是爭奪
• 建立獨特性，吸引機會
• 用槓桿放大努力
• 把工作變成遊戲

這就是「不靠運氣致富」的秘密。

— Naval Ravikant
