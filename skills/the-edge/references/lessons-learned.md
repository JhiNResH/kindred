# Lessons Learned

## 2026-01-27: Pistons vs Nuggets

### 情況
- Nuggets OUT: Jokic, Gordon, Johnson, Braun (4 starters)
- Pistons OUT: LeVert (1 role player)
- 推薦: 活塞

### 下注
- Pistons Spread -6.5: $6.75
- Red Wings ML: $4.50

### 結果
- Pistons 109 - Nuggets 107 (活塞贏 2 分)
- Spread -6.5 沒 cover ❌
- Red Wings 輸 ❌
- **總虧損: -$11.24**

### 問題
1. 下了 Spread 不是 Moneyline
2. 沒確認用戶下的是什麼
3. Red Wings 沒有傷病 edge

### 教訓
1. **傷病 edge = 會贏，不保證大勝**
2. **永遠推薦 Moneyline**
3. **下注前確認類型**
4. **只賭有 edge 的比賽**

### 如果用 Moneyline
- Pistons ML 會贏 ✅
- 假設賠率 55¢，$6.75 → ~$12.27
- 會賺 +$5.52 而不是虧 -$6.74
