# Sports Analysis Reference

## Moneyline vs Spread vs Total

| 類型 | 說明 | 何時用 |
|------|------|--------|
| **Moneyline** | 只要贏就行 | ✅ 傷病 edge |
| **Spread** | 要贏超過 X 分 | ❌ 風險太高 |
| **Total O/U** | 總分高/低 | 雙方都有傷兵時 |

## 為什麼不用 Spread

傷病優勢 = 對方會輸，但不代表會輸很多：
- 板凳球員會更努力
- 比賽節奏可能變慢
- 最後幾分鐘追分常見
- 垃圾時間會縮小分差

## 傷病等級影響

| 球員等級 | 影響 | Edge |
|----------|------|------|
| MVP (Jokic, Giannis) | 巨大 | ⭐⭐⭐⭐⭐ |
| All-Star | 大 | ⭐⭐⭐⭐ |
| Starter | 中 | ⭐⭐⭐ |
| Role Player | 小 | ⭐⭐ |
| Bench | 微乎其微 | ⭐ |

## 多人傷病組合

- 1 MVP OUT = 強 edge
- 2+ starters OUT = 很強 edge
- 3+ starters OUT = 極強 edge（如 2026-01-27 掘金）

## 分析 Checklist

```
□ 誰 OUT？(ESPN injuries)
□ 是 starter 還是 bench？
□ 對手傷病情況？
□ 主客場？
□ 背靠背？
□ 賠率有沒有反映傷病？
```

## 下注時機

- **最佳**: 傷病剛宣布，賠率還沒調整
- **好**: 比賽前 2-4 小時
- **風險**: Live betting（變數太多）
