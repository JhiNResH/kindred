## 💰 收入來源

### 1. 協議手續費 (Protocol Fee)

```
用戶 Swap $1000
    ↓
OmniPredict 收 0.1% = $1
    ↓
分配:
  • 50% → 國庫 (Treasury)
  • 30% → LP 獎勵
  • 20% → 團隊/開發
```

**預估**:
- 日交易量 $1M → 日收入 $1,000
- 年收入: ~$365K
- 隨規模增長

### 2. 收益分成 (Yield Share)

```
opUSDC 產生的收益:
  • 套利收益: $X
  • 做市收益: $Y
    ↓
協議抽成 10-20%
```

**例子**:
- TVL $10M，年化 12%
- 年收益 $1.2M
- 協議抽 15% = $180K

### 3. 聚合路由費 (Routing Fee)

```
跨市場最優執行:
  用戶省了 $50 (比單一市場更好價格)
      ↓
  協議收 $5 (節省的 10%)
      ↓
  用戶還是賺 $45
```

**價值主張**: "我幫你省錢，分我一點"

### 4. Premium 功能

```
免費版:
  • 基本聚合
  • 標準套利

Pro 版 ($50/月 或 質押 token):
  • 優先套利機會
  • 更低滑點
  • 私有策略
  • API 存取
```

### 5. B2B 授權

```
其他協議想整合預測市場?
  → 用 OmniPredict SDK
  → 支付授權費或交易分成
```

## 🏰 護城河 (Moats)

### 1. 流動性網絡效應

```
更多 LP → 更深流動性 → 更好價格
    ↓
更多用戶 → 更多交易量 → 更多 LP 收益
    ↓
更多 LP (循環)
```

**先發優勢**: 第一個做，累積最多流動性

### 2. 聚合數據優勢

```
越多交易 → 越多數據
    ↓
更好的:
  • 套利策略
  • 價格預測
  • 風險模型
    ↓
更好的收益 → 更多用戶
```

### 3. 整合深度

```
整合越多預測市場:
  Opinion ✓
  Polymarket ✓
  Kalshi ✓
  Azuro ✓
  ...
    ↓
競爭者要追趕需要時間
```

### 4. 品牌 & 信任

```
"預測市場收益 = OmniPredict"

用戶信任 → 選擇我們
    ↓
安全運營 → 更多信任 (循環)
```

## 💡 永續經營關鍵

### 1. 不依賴 Token 價格
```
收入來自:
  ✅ 手續費 (真實交易)
  ✅ 收益分成 (真實套利)
  ❌ Token 銷售
```

### 2. 成本控制
```
主要成本:
  • Gas (可轉嫁給用戶或從收益扣)
  • 伺服器 (Relayer/Oracle)
  • 開發人力

初期可以很精簡:
  • 1-2 人團隊
  • 雲端成本 <$500/月
  • Gas 由協議收入覆蓋
```

### 3. 社區驅動
```
開源核心代碼
    ↓
社區貢獻 = 免費開發
    ↓
降低成本 + 增加信任
```
