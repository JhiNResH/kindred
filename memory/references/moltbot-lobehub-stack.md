## 核心觀點

| Moltbot | LobeHub |
|---------|---------|

| 主動通知 | 多 agent 協作 |
| 單一 agent | Artifact 渲染 |

---

## LobeHub 功能

1. **Agent Groups** — 多 agent 協作（Moltbot 最大缺口）
2. **40+ Model Providers** — 多模型路由
3. **Knowledge Base / RAG** — 文件處理 + 向量搜尋
4. **Voice (TTS/STT)** — 語音整合
5. **Text-to-Image** — DALL-E 3, MidJourney
6. **Vision / Multimodal** — 圖片分析
7. **Artifacts** — 豐富渲染
8. **Branching** — 對話分支
9. **Agent Marketplace** — 預建 agents
10. **MCP Marketplace** — 10,000+ plugins

---

## 連接方式

**Bridge = MCP**

1. 在 LobeHub 設計 agent + 測試
2. 用同樣的 MCP servers 連接 Moltbot
3. 把最佳 system prompt 複製到 SOUL.md
4. Telegram 助手行為 = LobeHub agent
