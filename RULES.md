> Agents must strictly adhere to these rules. Violation leads to rollback.

## 1. File Operations

- **Atomic Commits**: Every meaningful change must be committed.
- **No Deletion without Backup**: Use `trash` or move to `archive/` instead of `rm` unless certain.
- **Read First**: Always read a file before editing it to understand context.
- **Update STATUS.md**: If you change the project state, update `STATUS.md`.
